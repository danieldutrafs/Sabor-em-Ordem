# 🎂 Sabor em Ordem

> **Status do Projeto:**  Em Desenvolvimento

## 📝 Objetivo do Software
O **Sabor em Ordem** nasceu de uma necessidade real: **ajudar minha mãe na gestão do seu negócio de salgados de festa.** O objetivo é profissionalizar o controle das encomendas dela, substituindo os cadernos de anotações por um sistema digital que organiza os pedidos, calcula o faturamento automático e garante que nenhuma entrega de centos de salgados seja esquecida.

## 🚀 Tecnologias Aplicadas
As tecnologias utilizadas refletem o aprendizado em desenvolvimento Java Desktop:
* **Linguagem:** Java 
* **Interface Gráfica:** Java Swing (GUI)
* **Persistência de Dados:** Hibernate / JPA
* **Banco de Dados:** MySQL
* **Gerenciamento de Dependências:** Maven
* **Controle de Versão:** Git & GitHub

## ✨ Funcionalidades do Sistema (Requisitos)
* **Gestão de Clientes:** Cadastro e busca dinâmica de clientes com visualização em cards.
* **Cadastro de Itens:** Registro de produtos com controle de quantidade em estoque.
* **Controle de Encomendas:** Registro completo de vendas, incluindo data de entrega, valor de entrada e tipo de saída (Retirada/Entrega).
* **Filtros de Status:** Visualização rápida de encomendas "Pendentes" e "Concluídas" diretamente na tela de encomendas.
* **Gestão de Estoque:** Baixa automática de produtos no estoque após a confirmação de uma venda.
* **Relatórios Financeiros:** Painel com faturamento total, número de pedidos realizados e lista de produtos mais vendidos.
* **Persistência Dinâmica:** Proteção de dados com transações "Tudo ou Nada" (Rollback em caso de erro).
